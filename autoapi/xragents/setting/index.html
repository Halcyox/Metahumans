class xragents.setting.InfiniteTelevision¶

This file holds functions that allow for the user to listen to ‘n’ number of AIs talk about things.

The user can also step in and break the fourth wall at any time under a chosen character.

Current limit is 9 supported animation nodes. However you can have unlimited text AI talking in a scene.
